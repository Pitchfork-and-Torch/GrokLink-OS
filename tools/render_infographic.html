What it is

GrokLink OS is a complete research operating system — kernel, HAL, drivers, services, GUI, and safety path owned end-to-end. It is not a Flipper overlay. Any compatible LLM can treat the device as a passive RF sensory peripheral while TX remains human-gated.

GrokLink RTOS Policy engine GrokAgent missions LLM tools ST7567 GUI RAM vault
Design principles
  • Safety by construction — actuators only via policy
  • USB-safe radio — deferred jobs, never in endpoint IRQs
  • Constrained autonomy — offline missions stay passive
  • Audit everything — append-only + host observation logs
  • Honest silicon — no false capability claims
